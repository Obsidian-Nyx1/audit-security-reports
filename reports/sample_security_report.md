# Sample Security Report

Asset inventory and log source onboarding are tracked for coverage.
Authentication events include login and failed login attempts for privileged access.
Timestamp normalization is enforced with NTP to reduce clock drift.
Logs are shipped with TLS forwarding and certificate validation.
Retention lifecycle includes archive and deletion based on legal hold rules.
Detection alert use case mapping follows MITRE coverage.
Incident triage includes ticket creation and MTTR tracking.
OWASP API authorization and request id logging are documented.
CVSS and CWE based vulnerability risk is linked to detections.
